Brenhin Keller (Jul 20 2021 at 06:10):

Yeah, so

((x,y),) -> (α*x, y)

is just fancy lambda syntax equivalent to

function (tpl)
    x,y = tpl
    return α*x, y
end

and the whole thing is equivalent to

function scalex(α)
    function (tpl)
       x,y = tpl
       return α*x, y
    end
end
