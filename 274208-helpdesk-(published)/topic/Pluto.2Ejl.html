I think to remember that I've read somewhere, that I can pause the reactivity temporarily, for example if I want to change multiple cells and each intermediate change would result in a corrupt state. Can someone point me to it? And are there ways to "mark" some successive cells as non-reactive so that the new evaluation only happens on manually triggering it afterwards (for example by a button)?

David Weingut (Feb 28 2021 at 17:41):

When I change a cell in Pluto I need to submit the change manually by Shift+Enter or pressing the play-like button. If I want to change multiple cells I just don't submit in between after I'm finished I press the "Submit all changes" button that can be found all the way near the top.

David Klaffenbach (Mar 04 2021 at 04:47):

Ctrl-S can be used to "Submit all changes".
