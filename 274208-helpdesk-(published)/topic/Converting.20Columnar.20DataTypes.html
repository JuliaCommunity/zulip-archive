qu bit (Mar 01 2021 at 06:53):

I am attempting to convert several columns in my dataframe that are
type String to type Float64. When I run the following code I am returning
the error message as:

begin
Attain = CSV.read("attainment.csv", header=true,
delim=",",df.DataFrame, types=Dict[4:11].=>Float64)
first(Attain,5)
end

Message:
MethodError: Cannot convert an object of type

Any suggestions?

Nils (Mar 01 2021 at 17:16):

The second error tells you what to do? You need to iterate over eachcol(Attain[!, 4:11]) is you want to map a function to a set of columns.

I used the following:

begin
ED4[!,7] = convert(Array{Float64,1}, ED4[!,7])
ED4[!,8] = convert(Array{Float64,1}, ED4[!,8])
ED4[!,9] = convert(Array{Float64,1}, ED4[!,9])
ED4[!,10] = convert(Array{Float64,1}, ED4[!,10])
ED4[!,11] = convert(Array{Float64,1}, ED4[!,11])
end

Perhaps there is a more elegant presentation?
